PROJECT TITLE: The HSAFX Console Example Pack

PURPOSE OF PROJECT: Provides some simple examples of how to use various 
features of the HSAFX console.

VERSION or DATE: January, 2019

HOW TO START THIS PROJECT: Right click a file in ConsoleTemplate or 
in the examples package, select "compile" and then "Run JavaFX Application".
Copy and rename ConsoleTemplate to create your own programs. See handouts 
in class for more information.
